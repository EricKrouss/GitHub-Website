Instructions On Creating A GitHub Website

Steps

  1. Create a GitHub account.
  2. Create a new repository
  3. Name it (REPLACE_WITH_YOUR_USERNAME).github.io
  4. Add in an index.html file with a heading and path to a photo along with a cooresponding image file
  5. Once you've uploaded the relevant files, click Commit Changes.
  6. Try testing to see if the site is up by searching for (REPLACE_WITH_YOUR_USERNAME).github.io in the URL box.
